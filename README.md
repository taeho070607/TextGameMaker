# TextGameMaker
텍스트게임을 만들경우 틀로서 작동할수있는 텍스트게임 엔진입니다.

게임을 제작하기위해서는 GameData//map 에 자신이 원하는 맵이릅.txt를 만들고 아래와 같은 양식으로 맵을 만들면 됩니다.
